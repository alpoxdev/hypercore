# Framework Reference

## Thiel 7 Questions

Score these areas:

- engineering
- timing
- monopoly
- people
- distribution
- durability
- secret

## Supporting validation lenses

- `YC PMF indicators`: demand pressure, organic pull, repeat behavior, downtime pain
- `The Mom Test`: rely on past behavior, not hypothetical praise
- `JTBD forces`: push, pull, habit, anxiety
- `Lean Canvas checks`: problem, segment, UVP, channels, revenue, unfair advantage

## Suggested synthesis

- start with Thiel scoring
- adjust confidence using PMF and evidence quality
- use JTBD and Mom Test logic to identify what to validate next
