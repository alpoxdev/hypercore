# Output Template

## Report sections

1. 문제 재진술
2. 프레임워크 선택 근거
3. 재정의된 기회 또는 HMW 프롬프트
4. 아이디어 세트
5. 근거 기반 우선순위화
6. 즉시 실행할 다음 단계

## Idea set format

각 아이디어에는 다음을 포함한다:

- 제목
- 짧은 설명
- 의미 있게 다른 점
- 작동할 수 있는 이유
- 가장 큰 위험

## Prioritization rule

- 관찰된 것과 추론한 것을 표시한다
- 상위 아이디어마다 그 뒤의 가정을 명명한다
- 다음으로 할 저비용 검증 단계를 추천한다
