# Formula Guide

Use only the frameworks that fit the task. Typical combinations:

- `JTBD + SCAMPER` for new idea generation
- `HMW + Morphological Box` for problem reframing
- `Six Thinking Hats + ERRC` for strategic options
- `TRIZ + inversion` for contradiction-heavy problems

## Selection heuristic

1. Pick one framing tool.
2. Pick one divergence tool.
3. Add one prioritization lens only if it changes the decision.

## Diversity checks

- different customer segment
- different channel or business model
- different cost structure
- different wedge or moat
