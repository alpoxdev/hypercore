# Formula Guide

작업에 맞는 프레임워크만 사용한다. 일반적인 조합:

- 새 아이디어 생성을 위한 `JTBD + SCAMPER`
- 문제 재정의를 위한 `HMW + Morphological Box`
- 전략적 선택지를 위한 `Six Thinking Hats + ERRC`
- 모순이 많은 문제를 위한 `TRIZ + inversion`

## Selection heuristic

1. 하나의 프레이밍 도구를 고른다.
2. 하나의 발산 도구를 고른다.
3. 결정이 달라질 때만 하나의 우선순위 렌즈를 추가한다.

## Diversity checks

- 서로 다른 고객 세그먼트
- 서로 다른 채널 또는 비즈니스 모델
- 서로 다른 비용 구조
- 서로 다른 웨지 또는 해자
