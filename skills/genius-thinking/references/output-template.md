# Output Template

## Report sections

1. Problem restatement
2. Framework selection rationale
3. Reframed opportunity or HMW prompt
4. Idea set
5. Evidence-backed prioritization
6. Immediate next steps

## Idea set format

For each idea include:

- title
- short description
- what makes it meaningfully different
- why it might work
- biggest risk

## Prioritization rule

- mark what is observed vs inferred
- name the assumption behind each top-ranked idea
- recommend the next cheap validation step
