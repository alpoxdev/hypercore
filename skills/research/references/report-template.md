# Report Template

Use this reference when you are drafting the saved research report.

## File Path

Save to:

`.hypercore/research/[NN].slug.md`

Example:

`.hypercore/research/03.websocket-vs-sse.md`

## Template

```markdown
# [Topic] Research Report

> Date reviewed: YYYY-MM-DD | Depth: quick/standard/deep | Sources reviewed: N | Sources cited: M

## Executive Summary
[Conclusion first. State the answer in 2-4 sentences.]

## Scope
- Question:
- Time window:
- Region or market:
- Evidence channels used:

## Key Findings
### 1. [Finding]
- Claim:
- Why it matters:
- Source: [Title](URL)

## Comparison
| Criteria | Option A | Option B | Notes |
|------|------|------|------|

## Risks And Caveats
- [Known limitation, conflict, or uncertainty]

## Recommendation
- [Decision or next action]

## References
- [Title](URL)
```

## Writing Rules

- Put the answer before the backstory.
- Link every non-obvious claim.
- Use exact dates when recency matters.
- Keep caveats explicit instead of hiding them in prose.
