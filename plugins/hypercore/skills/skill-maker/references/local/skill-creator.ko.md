# 로컬 skill-creator 요약

원본 출처:
- 설치된 시스템 `skill-creator` 스킬에서 캡처한 로컬 요약입니다.

`skill-maker`를 만들거나 고칠 때 이 참고 파일을 사용합니다.

이 요약은 자체적으로 읽히도록 작성되어 있습니다. 위 출처 메모는 과거 맥락일 뿐, 프로젝트 범위 작업 중 읽어야 하는 런타임 의존 경로가 아닙니다.

핵심 요약:

- metadata는 항상 컨텍스트에 들어오므로 스킬은 간결해야 합니다.
- 작업의 깨지기 쉬움에 따라 자유도를 조절해야 합니다.
- 검증은 평가 표면으로 다루고, 가능하면 현실적인 작업으로 시험해야 합니다.
- `SKILL.md`, rules, references, scripts, assets의 구조를 분명히 유지해야 합니다.
- 비대해진 코어 본문보다 점진적 공개를 선호해야 합니다.
- 에이전트 작업에 직접 도움이 되지 않는 부수 문서는 만들지 않는 편이 낫습니다.
