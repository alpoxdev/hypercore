## 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **일관성** | 같은 요소는 같은 스타일과 동작 |
| **계층 구조** | 중요한 것이 먼저 눈에 들어옴 |
| **단순함** | 색상 3-5개, 폰트 2-3개, 충분한 여백 |
| **접근성** | 충분한 대비, 읽기 쉬운 크기, 키보드 조작 |

## Quick Reference

### 색상 (60-30-10)
```
60% - 배경색 (중립색)
30% - 보조색 (카드, 섹션)
10% - 강조색 (버튼, CTA)
```

### 폰트 크기
```
h1: 32-48px | h2: 24-32px | h3: 18-24px
본문: 16-18px | 작은 텍스트: 12-14px
```

### 간격 (8px 단위)
```
xs: 4px | sm: 8px | md: 16px | lg: 24px | xl: 32px | 2xl: 48px
```
